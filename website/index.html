Frequently Asked Questions

Is Hootly.ai free to use?

Hootly.ai uses a BYOK (Bring Your Own Key) model — the extension is open-source with no subscription fees. You only pay for actual API usage directly to your LLM provider, with no middleman.

What is BYOK?

BYOK stands for "Bring Your Own Key." You provide your own API key from an AI provider (OpenAI, Anthropic, Google). This gives you full control over costs, no markup, and direct billing with your provider.

How do I get a Claude API key?

Go to console.anthropic.com, create an account, and generate an API key from your dashboard.

How do I get an OpenAI API key?

Visit platform.openai.com, sign up for an account, then create an API key in your account settings.

How do I get a Google Gemini API key?

Go to aistudio.google.com, sign in with your Google account, and create an API key.

How do I get an OpenRouter API key?

Visit openrouter.ai, create an account, and generate an API key. OpenRouter provides access to multiple AI models through a single API.

Where is my data stored?

All your data is stored locally in your browser. No data is sent to external servers except API calls to your chosen LLM provider.

Is my API key safe?

Your API key is stored locally in your browser's storage and is only sent to official API endpoints — never to any third-party servers.
