Frequently Asked Questions

How do I get a Claude API key?

Go to console.anthropic.com, create an account, and generate an API key from your dashboard.

How do I get an OpenAI API key?

Visit platform.openai.com, sign up for an account, then create an API key in your account settings.

How do I get a Google Gemini API key?

Go to aistudio.google.com, sign in with your Google account, and create an API key.

Where is my data stored?

All your data is stored locally in your browser. No data is sent to external servers except API calls to your chosen LLM provider.

Is my API key safe?

Your API key is stored locally in your browser's storage and is only sent to official API endpoints — never to any third-party servers.
